The GraphNode Structure

Every operation in the graph is represented by a GraphNode. This structure is optimized for cache locality and minimal memory overhead.

/* Graph Node Structure */ typedef struct GraphNode { int id; int opcode; int src1_id; // Index of parent node 1 (-1 if none) int src2_id; // Index of parent node 2 (-1 if none) int src3_id; // Index of parent node 3 (-1 if none) tensor_t *val; // Forward Value tensor_t *grad; // Backward Gradient int trainable; // 1 if this is a trainable parameter // Optimizer State (for Adam) tensor_t *m; // First moment tensor_t *v; // Second moment // Parameters (Scalars) double params[4]; // For operations that require a scalar parameter int heads; // For Multi-Head Attention int causal; // For Causal Masking int batch; // For Attention int seq; // For Attention int attn_type; // 0: Standard, 1: Linear Causal, 2: Linear Global } GraphNode;

Memory Management Strategy

  • Static Allocation: All intermediate tensors are allocated during graph_node creation. No allocations occur during the training loop.
  • Zero-Copy Gradients: Gradients are accumulated directly into the grad tensor of each node, avoiding redundant copies.
  • Optimizer State Persistence: For Adam, the m and v tensors persist across epochs, ensuring smooth convergence.

Optimizer Selection

The engine supports switching between optimizers globally. This affects the update step in graph_run.

OPTIMIZER_SGD (0)
W = W - LR * Grad
OPTIMIZER_ADAM (1)
Adaptive moments with bias correction.

Usage Examples (Ring)

Practical Implementation

1. Simple Linear Regression Graph

This example demonstrates how to build a single-layer graph for linear regression (Y = WX + B) and train it using the Graph Engine.

load "ringtensor.ring" # 1. Initialize Graph graph_init() # 2. Define Inputs and Weights x_data = tensor_init(1, 10) # Input w_vals = tensor_init(10, 1) # Weights y_target = tensor_init(1, 1) # Target # 3. Build Graph Nodes node_x = graph_node(OP_INPUT, 0, 0, x_data) node_w = graph_node(OP_WEIGHT, 0, 0, w_vals) node_matmul = graph_node(OP_MATMUL, node_x, node_w) node_target = graph_node(OP_INPUT, 0, 0, y_target) node_loss = graph_node(OP_MSE, node_matmul, node_target) # 4. Configure and Run Training graph_set_optimizer(OPTIMIZER_ADAM) graph_run(100, 0.01, 5.0) # 100 epochs, LR 0.01, cliping 5.0

2. Transformer Block Definition

Building a high-performance Transformer block using specialized kernels.

# Define a Transformer-style sequence node_emb = graph_node(OP_EMBEDDING, node_input, node_w_emb) node_ln = graph_node(OP_LAYERNORM, node_emb, node_w_ln) node_drop = graph_node(OP_DROPOUT, node_ln, 0) # Feed-Forward Network node_fc1 = graph_node(OP_MATMUL, node_drop, node_w1) node_relu = graph_node(OP_RELU, node_fc1, 0) node_fc2 = graph_node(OP_MATMUL, node_relu, node_w2) # Final Loss node_loss = graph_node(OP_MSE, node_fc2, node_target) graph_run(500, 0.001, 5.0)

Graph Engine API

High Performance
graph_init
Initializes or resets the global computational graph.
graph_init()
O(1)
Reset
Explanation

Clears all existing nodes and resets the graph counter to zero. Must be called before building a new model.

Ring Example
graph_init() # Ready for a new model
graph_node
Creates a new node in the graph. Returns Node ID.
graph_node(opcode, src1, src2, [tensor])
O(1)
Build
Explanation

Adds an operation or data node to the graph. src1 and src2 are IDs of parent nodes. For OP_INPUT or OP_WEIGHT, pass a Tensor object.

Ring Example
# Create a weight node w_node = graph_node(OP_WEIGHT, 0, 0, my_tensor) # Create an addition node add_node = graph_node(OP_ADD, node_a, node_b)
graph_run
Runs the full training loop entirely in C.
graph_run(epochs, lr)
O(E√óN)
Train
Explanation

Executes the training process. For each epoch, it performs a Forward pass, a Backward pass, and an Optimizer update for all trainable nodes.

Ring Example
# Train for 1000 epochs with 0.001learning rate graph_run(1000, 0.001)
graph_set_optimizer
Sets the optimizer: 0 (SGD) or 1 (ADAM).
graph_set_optimizer(OPTIMIZER_ADAM)
O(1)
Config
Explanation

Configures the global optimizer used by graph_run. Adam is recommended for Transformers.

Ring Example
graph_set_optimizer(OPTIMIZER_ADAM)
